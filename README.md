# StudyFolio

StudyFolio is a web-based productivity tool that rewards focused work with a virtual stock portfolio. Start a study session and watch your portfolio grow as long as you remain on the page. Ambient soundscapes help create a distraction-free environment.

## Features
- **Focus Timer**: Set a session length and track time spent studying or working.
- **Virtual Stock Portfolio**: The longer you focus, the more your virtual portfolio grows. Random dips mimic real market volatility.
- **Ambient Sounds**: Choose from Rainy Day, Coffee Shop, or Forest to help you concentrate. Sounds are generated with the Web Audio API so no downloads are required.
- **Stats Dashboard**: See total focus time and your best session, stored locally in the browser.

## Development
This project is a simple static site. Open `index.html` in a browser to try it out.

## Testing
There are no automated tests yet. Run `npm test` to execute a placeholder script.
